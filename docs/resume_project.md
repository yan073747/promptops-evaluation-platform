# Resume Project Description

## Short Version

PromptOps Evaluation Platform: Built a mock-first prompt evaluation platform with prompt version management, batch test runs, automatic scoring, failure case analysis, and cost/latency metrics.

## Bullet Points

- Designed a PromptOps workflow that manages prompt templates, test cases, model outputs, scoring results, and optimization history.
- Implemented a mock model mode so the project can run and be demonstrated without external API keys.
- Built automatic keyword-based scoring, human review fields, and prompt version comparison metrics including average score, failure rate, total cost, and average latency.
- Added structured failure analysis that records failed output text, severity, scoring reason, and prompt optimization suggestions.
- Prepared architecture notes, demo script, and interview explanation materials for portfolio presentation.

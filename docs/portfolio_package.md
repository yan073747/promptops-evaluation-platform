## 推荐投递标题

PromptOps 评测与 Prompt 优化平台

## 推荐简历一句话

基于 FastAPI + SQLite 实现 PromptOps 评测平台，支持 Prompt 版本管理、测试集管理、批量运行、自动评分、人工复核、失败案例分析、版本对比和成本耗时统计，并支持无 API Key 的 Mock 演示模式。

## 推荐面试讲法

这个项目展示的是 Prompt 工程化能力。我不是只写一个 Prompt 调模型，而是做了一套可测试、可评分、可复盘、可对比的 Prompt 评测流程。项目支持 Mock 模式，所以没有 API Key 也能演示完整链路；同时保留了接入真实模型 API 的扩展空间。

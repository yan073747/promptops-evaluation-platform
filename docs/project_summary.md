# 项目说明

PromptOps 评测与 Prompt 优化平台是一个面向 Prompt 工程化场景的技术展示项目。

它提供 Prompt 版本管理、测试集管理、Mock 批量运行、自动评分、人工复核、失败案例分析、版本对比、成本和耗时统计等能力。项目默认支持 Mock 模型模式，因此不依赖真实大模型 API Key，也能完整展示评测流程。

这个项目的核心目标是说明：Prompt 不应该只是一段临时文本，而应该像代码和配置一样被管理、测试、评估和持续优化。

## 技术要点

- 使用 FastAPI 提供 REST API。
- 使用 SQLite 存储 Prompt 版本、测试用例、运行记录、模型输出、评分结果和失败案例。
- 使用原生 HTML/CSS/JavaScript 构建可交互前端页面。
- 使用 Mock 模型保证项目可复现、可演示、可测试。
- 使用单元测试覆盖核心服务、数据存储和 API 行为。

## 可展示能力

- PromptOps 工作流设计能力。
- 后端接口与数据建模能力。
- 前端交互和可视化展示能力。
- 自动化测试和工程化交付能力。
- 对 AI 输出质量评估、失败分析、版本对比的理解。

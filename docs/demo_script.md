## 60-Second Version

1. Open the dashboard and explain that this is a PromptOps platform for evaluating prompt quality.
2. Click `重置演示数据` to start from a clean state.
3. Show the core workflow: prompt version, test set, batch run, automatic score, failure review, and optimization.

5. Show batch results and explain that each test case receives a score.

## Interview Explanation

This project demonstrates that I understand prompt engineering as an engineering process, not just prompt writing. I built a workflow that manages prompt versions, runs test cases in batches, scores outputs, records failures, and compares improvements.

07 / Workflow

评测流程

  1. 选择 Prompt 版本和测试集，建立可复用的评测任务。
  2. 选择不同版本运行评测，观察平均分、失败率、成本和耗时变化。

  4. 补充人工评分，让人审与自动评分一起沉淀评估记录。
